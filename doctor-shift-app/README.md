# 🏥 医師シフト管理システム

医師のシフトを自動生成・管理するWebアプリです。Streamlitで動作します。

## 機能一覧

| 機能 | 説明 |
|------|------|
| 👨‍⚕️ 医師設定 | 医師の登録・NG日・希望日・優先度などの制約設定 |
| 📊 シフト制約CSV | シフト種別ごとの月間最大回数をCSVで一括設定 |
| 📌 固定シフト入力 | 手動入力シフトを絶対反映（残りをAI自動生成） |
| 📅 シフト生成・表示 | 月ごとのシフト自動生成・CSV出力 |
| 📈 統計・分析 | 割当回数の可視化・制約チェック |

## シフト種別

| 曜日 | シフト枠 |
|------|----------|
| 平日 | 宿直A、宿直B、外来宿直 |
| 休日・祝日 | 宿直A、宿直B、外来宿直、日直A、日直B、外来日直 |

## 制約条件（医師ごとに設定可能）

- **NG日付**：指定日はシフトに入れない
- **希望日**：優先的にシフトを割り当てる
- **希望優先度**：希望日の重み付け（1〜5）
- **最低空ける日数**：連続勤務防止（例：2日空ける）
- **月間最小・最大回数**：月あたりの勤務回数の下限・上限
- **優先度**：高いほど優先的にシフトを割り当てる（1〜10）

## ローカルで起動する

```bash
# 1. リポジトリをクローン
git clone https://github.com/YOUR_USERNAME/doctor-shift-app.git
cd doctor-shift-app

# 2. 依存パッケージをインストール
pip install -r requirements.txt

# 3. アプリを起動
streamlit run app.py
```

ブラウザで `http://localhost:8501` を開いてください。

## Streamlit Cloudへのデプロイ

1. このリポジトリをGitHubにプッシュ
2. [share.streamlit.io](https://share.streamlit.io) にアクセス
3. "New app" → リポジトリを選択 → `app.py` を指定
4. "Deploy" をクリック

デプロイ後は自動で公開URLが発行されます。

## CSVフォーマット（シフト別上限回数）

アプリ内「📊 シフト制約CSV」ページでテンプレートをダウンロードできます。

```csv
医師名,宿直A,宿直B,外来宿直,日直A,日直B,外来日直
田中 一郎,4,4,3,2,2,2
鈴木 花子,3,3,4,2,2,2
```
